## SECTION A: Fundamentals of System Security (P1)

### 1. System security mainly focuses on:
A. Faster systems  
B. Protecting systems and data  
C. UI design  
D. Hardware speed  

✅ **Answer: B**

---

### 2. Which is NOT a security goal?
A. Confidentiality  
B. Integrity  
C. Availability  
D. Performance  

✅ **Answer: D**

---

### 3. Confidentiality means:
A. Data is always available  
B. Data is accurate  
C. Data is hidden from unauthorized users  
D. Data is backed up  

✅ **Answer: C**

---

### 4. Integrity ensures:
A. Data is secret  
B. Data is correct and unchanged  
C. Data is fast  
D. Data is encrypted  

✅ **Answer: B**

---

### 5. Availability means:
A. Data is encrypted  
B. Data is accessible when needed  
C. Data is hidden  
D. Data is duplicated  

✅ **Answer: B**
