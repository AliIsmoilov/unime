## PART 1: Multiple Choice Questions (MCQs)

### 1. What is the main role of an Operating System in security?
A. Speed up programs  
B. Protect system resources  
C. Improve internet speed  
D. Design applications  

✅ **Answer: B**

---

### 2. Which OS resource must be protected to prevent programs from accessing each other’s data?

B. Memory  
C. Printer  
D. Network cable  

✅ **Answer: B**

---

### 3. Which design principle divides OS into layers?
A. Modular design  
B. Layered design  
C. Virtual design  
D. Shared design  

✅ **Answer: B**

---

### 4. What is virtualization mainly used for?
A. Faster execution

✅ **Answer: C**

## PART 2: Short Q&A (Very Exam-Friendly)

### Q1. What does an OS protect?
**A:** Memory, files, programs, devices, and network resources.

---

### Q2. What is layered OS design?
**A:** OS is built in layers, each with a specific responsibility.

---

### Q3. What is virtualization?
**A:** Creating isolated virtual machines on one physical system.

---

### Q4. Why is memory protection important?
**A:** Prevents programs from accessing unauthorized data.

---

### Q5. What is a kernel?
**A:** The core part of the OS that controls system resources.

---

### Q6. What is a reference monitor?
**A:** A mechanism that checks every access to resources.

---

### Q7. What is a trusted system?
**A:** A system proven to enforce its security policy correctly.

---

### Q8. What is a rootkit?
**A:** Malware that hides in the OS with root/admin access.
