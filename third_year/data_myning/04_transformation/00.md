# PCA and Hough Transform — Short Notes

# 1. Dimensionality Reduction and PCA

## Problem
High-dimensional data is difficult to:
- visualize
- analyze
- find correlations in

Example:
- Dataset with 53 blood test features

---

# PCA (Principal Component Analysis)

PCA reduces the number of dimensions while keeping the most important information.

---

# Main Goals of PCA

- Reduce feature count
- Preserve important information
- Simplify data
- Improve visualization
- Speed up machine learning

---

# How PCA Works

PCA creates new axes called:
- Principal Components (PCs)

These components:
- capture the largest variance in the data.

---

# What is Variance?

Variance shows:
- how spread out the data is.

PCA tries to:
- keep the directions with maximum variance.

---

# Principal Components

- PC1 → largest variance
- PC2 → second largest variance
- and so on

Formula:

```math
\lambda_1 \geq \lambda_2 \geq \lambda_3 \geq \cdots \geq \lambda_n
```

Where:
- larger eigenvalue = more important component

---

# What is an Eigenvalue?

Eigenvalue measures:
- how much information a component keeps.

Large eigenvalue:
- more variance
- more information

Small eigenvalue:
- less useful information

---

# Feature Selection in PCA

PCA:
- computes covariance matrix
- finds eigenvectors and eigenvalues

Then:
- keeps only components with the largest eigenvalues.

---

# Advantages of PCA

✅ Reduces data size  
✅ Faster computation  
✅ Easier visualization  
✅ Removes correlation  

---

# Disadvantages of PCA

❌ Some information is lost  
❌ Original measurement units disappear  
❌ Harder to interpret  

---

# Why Original Units Disappear

PCA creates new features using combinations of original features.

Example:

```math
PC1 = 0.5 \times Height + 0.7 \times Weight - 0.2 \times Time
```

Here:
- meters
- kilograms
- seconds

are mixed together.

So:
- the new component no longer has a clear physical unit.

---

# PCA Formula

```math
PC = w_1x_1 + w_2x_2 + \cdots + w_nx_n
```

Where:
- \(x_i\) → original feature
- \(w_i\) → weight

---

# Why Original Features Are Needed Before PCA

PCA:
- creates new components from original features.

So:
1. Original data must exist first
2. Then PCA reduction is applied

---

# 2. Hough Transform

## What is Hough Transform?

Hough Transform is an:
- image processing
- computer vision

technique used to detect:
- lines
- circles
- ellipses

and other shapes.

---

# Main Idea

Hough Transform works using a:
- voting system

Each edge point:
- votes for possible shapes.

The shape with the most votes:
- is detected as the real shape.

---

# How Hough Transform Works

## 1. Edge Detection

First:
- edges are extracted from the image.

Usually:
- Canny Edge Detector is used.

---

## 2. Voting

Each edge point:
- votes in parameter space.

Votes are stored in:
- accumulator array.

---

## 3. Peak Detection

The highest peak:
- represents the detected shape or line.

---

# Polar Representation

Normal line equation:

```math
y = mx + b
```

Hough Transform uses:

```math
\rho = x\cos\theta + y\sin\theta
```

---

# Benefits of Polar Representation

✅ Handles vertical lines correctly  
✅ More stable representation  
✅ Easier computation  

---

# How Noise Affects Hough Transform

Noise:
- makes peaks fuzzy
- spreads votes
- reduces accuracy

---

# Bin Size Problem

## Small bins
✅ High precision  
❌ Sensitive to noise  

---

## Large bins
✅ Noise tolerant  
❌ Lower precision  

---

# Advantages of Hough Transform

✅ Robust to noise  
✅ Detects incomplete shapes  
✅ Handles outliers well  

---

# Disadvantages of Hough Transform

❌ High memory usage  
❌ Slow computation  
❌ Difficult bin-size selection  

---

# Circle Detection in Hough Transform

## Unknown Radius

Uses 3D parameter space:

- a → center x
- b → center y
- r → radius

---

## Fixed Radius

Uses 2D parameter space:

- a → center x
- b → center y

---

# Important Note

As the number of model parameters increases:
- computation cost increases
- memory usage increases

Because:
- accumulator grid size grows exponentially.

---

# Overall Summary

## PCA
- reduces dimensionality
- keeps the most important variance
- simplifies data

---

## Hough Transform
- detects shapes in images
- uses a voting mechanism
- works well with noise and occlusion
