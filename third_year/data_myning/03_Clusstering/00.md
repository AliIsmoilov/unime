# 🧠 Unsupervised Learning: Clustering

## 1. What is Clustering?
Clustering is an **unsupervised learning** method used to group similar data points together.

### Main Goal
- ✅ **High similarity inside a cluster** → points in the same group should be close.
- ✅ **High difference between clusters** → different groups should be far apart.

### Easy Memory Trick
> **"Close friends stay together, strangers stay apart."**

---

# 2. Types of Clustering

| Method | Simple Idea |
|---|---|
| **Partitioning** | Split data into separate groups |
| **Hierarchical** | Build clusters like a family tree |
| **Density-Based** | Find dense areas and ignore noise |
| **Centroid-Based** | Group around center points |
| **Model-Based** | Assume data follows probability models |

---

# 3. K-Means Clustering

## What is K-Means?
K-Means divides data into **K clusters** using **centroids (center points)**.

### How it Works
1. Choose number of clusters (**K**)
2. Place random centroids
3. Assign points to nearest centroid
4. Update centroid positions
5. Repeat until stable

### Distance Used
Usually uses **Euclidean Distance**.

---

## Finding Best K

### 1. Elbow Method
- Plot **K vs Inertia**
- Choose the "elbow point"

📌 Memory Tip:
> "Where improvement suddenly slows = best K"

### 2. Silhouette Score
Measures cluster quality.

| Score | Meaning |
|---|---|
| +1 | Perfect clustering |
| 0 | Overlapping clusters |
| -1 | Wrong clustering |

---

# 4. K-Means Advantages & Disadvantages

## ✅ Advantages
- Fast and efficient
- Works well on large datasets
- Simple to understand

## ❌ Disadvantages
- Need to choose K beforehand
- Sensitive to outliers/noise
- Can get stuck in local optimum
- Doesn't work well with categorical data
- Poor for non-circular shapes

📌 Memory Tip:
> "K-Means likes round and clean clusters."

---

# 5. Hierarchical Clustering

Creates a tree structure called a **Dendrogram**.

---

## Types

### 1. Agglomerative (Bottom-Up)
- Start with individual points
- Merge closest clusters step by step

📌 Memory Tip:
> "Small → Big"

---

### 2. Divisive (Top-Down)
- Start with one large cluster
- Split into smaller clusters

📌 Memory Tip:
> "Big → Small"

---

# 6. Linkage Methods

Linkage decides how distance between clusters is measured.

| Linkage | Measures |
|---|---|
| **Single Linkage** | Closest points |
| **Complete Linkage** | Farthest points |
| **Average Linkage** | Average distance |
| **Centroid Linkage** | Distance between centers |

# 🚀 Quick Revision

## Clustering Goal
- Minimize intra-cluster distance
- Maximize inter-cluster distance

## K-Means
- Needs K
- Uses centroids
- Fast but sensitive to noise

## Hierarchical
- Builds dendrogram
- Agglomerative = merge
- Divisive = split

## Linkage
- Single = nearest
- Complete = farthest
- Average = mean
- Centroid = center
