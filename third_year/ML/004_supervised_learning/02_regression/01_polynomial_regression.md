# Polynomial Regression

Polynomial Regression is a type of regression used when the relationship between input and output is **curved instead of straight**.

---

# Linear Regression vs Polynomial Regression

## Linear Regression (Straight Line)

\[
y = mx + b
\]

Example graph:

```text
y
↑
|        *
|      *
|    *
|  *
|*
+----------------→ x
```

---

## Polynomial Regression (Curved Line)

\[
y = ax^2 + bx + c
\]

Example graph:

```text
y
↑
|          *
|       *     *
|     *         *
|   *             *
| *                 *
+----------------------→ x
```

---

# Simple Explanation

- **Linear Regression** fits a straight line.
- **Polynomial Regression** fits a curve.

It is useful when data does not follow a straight pattern.

---

# Example

Suppose:
- Studying 1 hour increases marks slowly
- Studying more hours increases marks faster

This creates a curved relationship, so Polynomial Regression works better.

---

# Real-Life Uses

- Population growth prediction
- Stock market trends
- Temperature changes
- Sales forecasting
