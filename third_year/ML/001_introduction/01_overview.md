# What is Machine Learning?

Machine Learning(ML) is a branch of Artificial Intelligence(AI) that enables systems to learn from data and improve their performance without being programmed for every task.

Instead of following fixed instructions, ML algorithms identify patterns in data and make predictions or decisions automatically.

## Example
Retail websites use Machine Learning algorithms to analyze customer behavior and purchase history in order to provide personalized product recommendations.


# Machine Learning Types/Categories

## 1. Supervised Learning

Supervised learning uses labeled data, where the correct output is already known. The model learns by mapping inputs to the correct outputs.

**Example:** Email spam detection.

**Algorithms:**  
- Regression algorithms  
- Classification algorithms  
- Naïve Bayes classifiers
- Neural networks
- Random forest algorithms  

---

## 2. Unsupervised Learning

Unsupervised learning works with unlabeled data. The model identifies hidden patterns, relationships, or groups within the data.

**Example:** Customer segmentation in marketing.

**Algorithms:**  
- K-Means Clustering  
- Hierarchical Clustering  
- Probabilistic clustering

---

## 3. Semi-Supervised Learning

Semi-supervised learning combines a small amount of labeled data with a large amount of unlabeled data to improve learning accuracy.

**Example:** Medical image classification with limited labeled images.

**Algorithms:**  
- Semi-Supervised SVM  
- ...

---

## 4. Self-Supervised Learning

Self-supervised learning creates labels from the data itself instead of requiring massive anotated or labeled datasets. The model create labels and teaches yourself using provided data.

**Example:** Predicting missing words in a sentence.

---

## 5. Reinforcement Learning

Reinforcement learning, also called reinforcement learning from human feedback(RLHF), is a type of learning that trains algorithms using a system of reward and punishment/penalties.
The agent is rewarded or penalized for it's actions based on established metric/points, encouragin agent to continue good practices and discard bad ones. With repetions, the agent learns best strategies.

**Example:** Common in video games and teach robots how to replicate human tasks.
