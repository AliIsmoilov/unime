# 🧩 Java Enum 

## ✅ What is `enum` in Java?

`enum` (short for "enumeration") is a special **data type** in Java that allows you to define a set of **constant values**.

You can use it when you have **fixed options** that don't change — like days of the week, colors, directions, etc.

---

## 🔤 Syntax of `enum`

```java
enum Day {
    MONDAY, 
    TUESDAY,
    WEDNESDAY, 
    THURSDAY, 
    FRIDAY, 
    SATURDAY, 
    SUNDAY
}
```
